# frontend-monthly

## local preview

```
cd site
```

```
npm ci
```

```
npm run start
```

open `http://localhost:8000`

## add post

create md file into the `/posts` dir.

## add profile image

add image({userName}.jpg) file into the `/static` dir.

## publish

merge PR or push to `master` branch then starting auto publish by GitHub Actions.
